#C++ sort()排序

##例题一：
题目描述

某涉密单位下发了某种票据，并要在年终全部收回。每张票据有唯一的ID号。全年所有票据的ID号是连续的，但ID的开始数码是随机选定的。因为工作人员疏忽，在录入ID号的时候发生了一处错误，造成了某个ID断号，另外一个ID重号。你的任务是通过编程，找出断号的ID和重号的ID。假设断号不可能发生在最大和最小号。

输入格式
第一行包含整数 N，表示后面共有 N 行数据。接下来 N 行，每行包含空格分开的若干个（不大于100个）正整数（不大于100000），每个整数代表一个ID号。

输出格式
要求程序输出1行，含两个整数 m,n，用空格分隔。其中，m表示断号ID，n表示重号ID。

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    int N;
    cin >> N;

    vector<int> ID;
    int temp;

    while (cin >> temp) {//输出的时候在输入完数据之后要按Ctrl+Z，然后回车，就可以输出结果了
        ID.push_back(temp);
    }

    sort(ID.begin(), ID.end());

    int m = -1; // 初始化m为一个无效值
    int n = -1; // 初始化n为一个无效值

    for (int i = 0; i < ID.size() - 1; i++) {
        if (ID[i + 1] - ID[i] > 1) {
            m = ID[i] + 1;
        }
        if (ID[i] == ID[i + 1]) {
            n = ID[i];
        }
    }

    cout << m << " " << n;

    return 0;
}

}

##例题二：
题目描述

某次科研调查时得到了几个自然数，每个数均不超过1.5<10^9。已知不相同的数不超过10^4个，现在需要统计这些自然数各自出现的次数，并按照自然数从小到大的顺序输出统计结果。

输入描述
第1行是整数，表示自然数的个数。第2~n+1行每行一个自然数。其中1≤n≤2×10^5，每个数均不超过1.5≤10^9

输出描述
输出m行(m为n个自然数中不相同数的个数)，按照自然数从小到大的顺序输出。每行输出两个整数，分别是自然数和该数出现的次数，其间用一个空格隔开。

#include <bits/stdc++.h>
using namespace std;
int n;
vector <int> number;
int temp;
int c=0;
int main()
{ cin>>n;
for(int i=0;i<n;i++)
{ cin>>temp;
  number.push_back(temp);
}

sort(number.begin(),number.end());
for(int i=0;i<n;i++)
  {   c++;
        if(number[i]!=number[i+1])//当下一个数不和当前数相等时说明重复数结束
         {
           cout<<number[i]<<" "<<c<<endl;
           c=0;
         }
  }
  return 0;
}

例题三#

设有几个正整数a1..a_n,将它们联接成一排，相邻数字首尾相接，组成一个最大的整数。
输入描述
第一行有一个整数，表示数字个数n。第二行有n个整数，表示给出的n个整数a_i.其中，1≤n≤20,1≤a_i≤10^9。
输出描述
输出一个正整数，表示最大的整数

#include <bits/stdc++.h>
using namespace std;
int n;
string temp;
vector<string> number;
bool cmp(string a,string b)
{
    if(a+b>b+a)   //如果前面的大，则第一个参数在第二个参数之前，实现了降序排序
      return true;
    else
      return false;
}
int main()
{  cin>>n;
    for(int i=0;i<n;i++)
    { cin>>temp;
        number.push_back(temp);
    }
    sort(number.begin(),number.end(),cmp);

   for(auto i:number)
   {
       cout<<i;
   }
    return 0;
}

通过这题我想来介绍一下c++里面的sort（）函数的细节。
首先sort（）里面一般有三个参数，第一个是排序的起点，第二个则是终点，这两个参数类型都必须是迭代器。第三个参数是比较参数，决定排序的方式。
这里c++官方技术文档是这样定义的：
第一个和第二个参数：随机访问要排序的序列的初始位置和最终位置的迭代器。使用的范围为，包含first和last[first,last)之间的所有元素，包括first指向的元素但不包括last指向的元素。随机访问迭代器应指向一个类型交换已正确定义，并且既可移动构造又可移动分配。
第三个参数： 接受范围内的两个元素作为参数并返回可转换为 的值的二元函数bool。返回的值指示作为第一个参数传递的元素是否被视为在它定义的特定严格弱排序中位于第二个参数之前。该函数不得修改其任何参数。这可以是函数指针或函数对象。

这里不用管严格弱排序的意思，这句话换句话的意思就是说，如果返回值是True，则第一个参数将会在第二个参数前面，返回值是True则反之。
拿本题的例子来说：

           bool cmp(string a,string b)
          {
                if(a+b>b+a)   //如果前面参数1大，则第一个参数会第二个参数之前，总体上来看就是大的会在小的前面，这就相当于实现了降序排序。
                  return true;
                else
                  return false;
                } //这里的a就代表了参数1，这里的b就代表了参数2.

例题一#

某小学最近得到了一笔赞助，打算拿出其中一部分为学习成绩优秀的前 5 名学生发奖学金。期末，每个学生都有 3 门课的成绩：语文、数学、英语。先按总分从高到低排序，如果两个同学总分相同，再按语文成绩从高到低排序，如果两个同学总分和语文成绩都相同，那么规定学号小的同学排在前面，这样，每个学生的排序是唯一确定的。
任务：先根据输入的 3 门课的成绩计算总分，然后按上述规则排序，最后按排名顺序输出前 5 名学生的学号和总分。注意，在前 5 名同学中，每个人的奖学金都不相同，因此，你必须严格按上述规则排序。例如，在某个正确答案中，如果前两行的输出数据（每行输出两个数：学号、总分）是：

          7 279
        5 279

这两行数据的含义是：总分最高的两个同学的学号依次是 7 号、5 号。这两名同学的总分都是 279 （总分等于输入的语文、数学、英语三科成绩之和），但学号为 7 的学生语文成绩更高一些。如果你的前两名的输出数据是：

            5 279
          7 279

则按输出错误处理，不能得分。
输入描述
第1行为一个正整数n(6≤n≤300)，表示该校参加评选的学生人数。第2到n+1行，每行有3个用空格隔开的数字，每个数字都在0到100之间。第j行的3个数字依次表示学号为j-1的学生的语文、数学、英语的成绩。每个学生的学号按照输入顺序编号为1~n(恰好是输入数据的行号减1)。所给的数据都是正确的，不必检验。
输出描述
输出共有5行，每行是两个用空格隔开的正整数，依次表示前5名学生的学号和总分。

源代码#

#include <bits/stdc++.h>
using namespace std;
struct score
{ int index;
  int chinese;
  int math;
  int english;
  int sum;
}s[305];
bool cmp(struct score a,struct score b)
{  if(a.sum==b.sum)
    {   if(a.chinese==b.chinese)
        {   if(a.index<b.index)
                return true;
            else
                return false;
        }
         else if(a.chinese>b.chinese)
            return true;
         else
            return false;
     }
    else if(a.sum>b.sum)
        return true;
    else
        return false;
}
int main()
{int n;
 cin>>n;
 for(int j=1;j<=n;j++)
 {   s[j].index=j;
   cin>>s[j].chinese>>s[j].math>>s[j].english;
   s[j].sum=s[j].chinese+s[j].math+s[j].english;
 }
 sort(s+1,s+1+n, cmp);
 for(int i=1;i<=5;i++)
 {
     cout<<s[i].index<<" "<<s[i].sum<<endl;
 }
  return 0;
}

例题二#

外卖店优先级
题目描述
“饱了么”外卖系统中维护着 N 家外卖店，编号 1 ∼ N。每家外卖店都有 一个优先级，初始时 (0 时刻) 优先级都为 0。每经过 1 个时间单位，如果外卖店没有订单，则优先级会减少 1，最低减 到 0；而如果外卖店有订单，则优先级不减反加，每有一单优先级加 2。如果某家外卖店某时刻优先级大于 5，则会被系统加入优先缓存中；如果 优先级小于等于 3，则会被清除出优先缓存。给定 T 时刻以内的 M 条订单信息，请你计算 T 时刻时有多少外卖店在优 先缓存中?
输入描述
第一行包含 3 个整数 N,M,T。以下 M 行每行包含两个整数 ts,id，表示 ts 时刻编号 id 的外卖店收到一个订单。其中，1≤N,M,T≤10^5 ,1≤ts≤T，1≤id≤N。
输出描述
输出一个整数代表答案。

#include <bits/stdc++.h>
using namespace  std;
int M,N,T;
int locate[2000000]={0};//定位上一次各个的订单时间
int prior[2000000]={0};//存储各个店的优先级
int flag[2000000]={0};//标记是否满足优先级>5
struct store
{ int id;
   int ts;
}store[200000];
bool  cmp(struct store a,struct store b)
{
    if(a.ts==b.ts)
    {
        return a.id<b.id;
    }
    return a.ts<b.ts;
}
int main()
{ cin>>N>>M>>T;
   for(int i=0;i<M;i++)
   {
       cin>>store[i].ts>>store[i].id;
   }
    sort(store,store+M, cmp);
   for(int i=0;i<M;i++)
   {  if(store[i].ts!=locate[store[i].id])
       {
        prior[store[i].id]= prior[store[i].id]-(store[i].ts-locate[store[i].id]) +1;
       }//如果当前数组的时刻与当前数组对应的ID店的上一次订单时间不相等，就减去对应相差的时间。
       prior[store[i].id]= prior[store[i].id]<0?0: prior[store[i].id];//防止优先级<0
   
       if(prior[store[i].id]<=3)
       {
         flag[store[i].id]=0;
       }//标记优先级
       prior[store[i].id]+=2;//优先级+2
       if(prior[store[i].id]>5)
       {
           flag[store[i].id]=1;
       }//标记优先级
       locate[store[i].id]=store[i].ts;//当前时间判断完了，就将当前时间设置为该ID店的最后一次订单时间。用于下一次比较
   }
 for(int i=1;i<=N;i++)
 {  if(locate[i]<T)
     {
         prior[i]= prior[i]-(T-locate[i]);
     }
    if(prior[i]<=3)
        flag[i]=0;
 }
 int ans=0;
 for(int i=1;i<=N;i++)
 {
     if(flag[i])
         ans++;
 }
cout<<ans;
}

这里我非常有必要解释一下为什么prior[store[i].id]= prior[store[i].id]-(store[i].ts-locate[store[i].id]) +1;这段代码要+1。我们知道这段代码的功能其实是想要求出两段时间相差了多少，比如6-9中间相差了2个数字，我们直接9-6求出来是等于3的，这个3其中也把9算了进去，但是由于我们到达9的时候是要加优先级的，所以我们要把多减掉的一位给它加回来
